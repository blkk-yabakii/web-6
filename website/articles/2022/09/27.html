URL

URL (Uniform Resource Locator) merupakan format untuk mendeskripsikan lokasi file di internet.

Perhatikan URL berikut

https://www.google.com/search?q=balai+latihan+kerja+komunitas

URL tersebut tersusun dari beberapa komponen, yaitu:

https://
protocol untuk menunjukkan metode transfer data melalui jaringan.
www.google.com
domain untuk menunjukkan alamat server
/search
path untuk menunjukkan lokasi file di server
?q=balai+latihan+kerja+komunitas
query untuk menunjukkan informasi tambahan yang diperlukan oleh server untuk diproses lebih lanjut.
