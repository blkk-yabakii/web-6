Javascript

Javascript adalah salah satu bahasa pemrograman. Berbeda dengan HTML ataupun CSS yang merupakan bahasa markup. Artinya, dengan Javascript sebagai bahasa pemrograman, yang dipelajari adalah penyusunan logika komputer.

Javascript sangat umum digunakan, terutama di website ataupun aplikasi mobile. Biasanaya, Javascript dijalankan bersandingan dengan HTML dan CSS yang mana dijalankan oleh browser semacam Firefox, Google Chrome ataupun Edge. Tapi, Javascript juga bisa dijalankan tanpa browser, yaitu menggunakan NodeJS, untuk pemrograman server atau aplikasi berbasis desktop.

Untuk memulai program Javascript di browser, yang pertama dilakukan adalah dengan membuat link ke Javascript dari file HTML. Contoh: 

<html>
  <head>
    <script charset="UTF-8" type="application/javascript" src="path/ke/lokasi/javascript.js"></script>
  </head>
</html>

Statement

Program javascript dieksekusi baris perbaris. Artinya, dalam memproses file javascript, browser akan membaca baris-perbaris source code dan langsung dieksekusi seketika itu juga. Contoh:

const a = 1;
const b = 2;
const c = a + b;
alert(c);

Program tersebut akan menampilkan dialog dengan isi 3.

Setiap baris program dalam javascript ditutup dengan tanda ;. Dalam peristilahan pemrograman, baris code yang dieksekusi oleh program disebut dengan istilah statement.

Output

Output atau hasil program javascript dapat ditampilkan dengan berbagai cara. Untuk pemula, agar lebih muda diobservasi, umumnya output javascript dilakukan dengan menggunakan alert sebagaimana di atas. Akan tetapi, penggunaan alert sebagaimana di atas dapat mengganggu penggunaan website.

Cara lain yang lebih tidak mengganggu adalah menggunakan console.log. Contohnya:

const a = 1;
const b = 2;
const c = a + b;
console.log(c);

Untuk melihat hasil program di atas, harus membuka developer tools terlebih dahulu.

Variables

Variable digunakan untuk memberi nama pada suatu data agar mudah diingat oleh penulis program. Pembuatan variable dapat dilakukan dengan tiga cara, yaitu:

  • const c = 3;
  • let b = 2;
  • var a = 1;

Perbedaannya, const digunakan untuk menamai variable yang tidak akan diganti lagi. Contohnya:

const a = 1;
a = 10;

Program di atas akan error, karena program tersebut mencoba merubah variable a yang asalnya berisi 1 menjadi 10. Berbeda dengan let atau var, keduanya mengizinkan perubahan data/isi variable. Contoh:

let a = 1;
var b = 2;
a = 10;
b = 20;

Program tersebut dapat berjalan tanpa masalah.

Karena const tidak dapat dirubah, maka disarankan untuk menggunakan const selama tidak ada rencana perubahan isi variable.

Operator

Javascript menyediakan beberapa operator untuk operasi aritmetika sederhana:

Operator Fungsi Contoh Output
+ Penjumlahan 
const a = 1;
const b = 2;
console.log(a + b);
 3
- Pengurangan 
const a = 1;
const b = 2;
console.log(a - b);
 -1
* Perkalian 
const a = 4;
const b = 2;
console.log(a * b);
 8
/ Pembagian 
const a = 4;
const b = 2;
console.log(a / b);
 2
% Sisa pembagian 
const a = 4;
const b = 3;
console.log(a % b);
 1
