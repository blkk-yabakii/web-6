Javascript

Javascript adalah salah satu bahasa pemrograman. Berbeda dengan HTML ataupun CSS yang merupakan bahasa markup. Artinya, dengan Javascript sebagai bahasa pemrograman, yang dipelajari adalah penyusunan logika komputer.

Javascript sangat umum digunakan, terutama di website ataupun aplikasi mobile. Biasanaya, Javascript dijalankan bersandingan dengan HTML dan CSS yang mana dijalankan oleh browser semacam Firefox, Google Chrome ataupun Edge. Tapi, Javascript juga bisa dijalankan tanpa browser, yaitu menggunakan NodeJS, untuk pemrograman server atau aplikasi berbasis desktop.

Javascript di HTML

Untuk memulai program Javascript di browser, yang pertama dilakukan adalah dengan membuat link ke Javascript dari file HTML. Contoh: 

<html>
  <head>
    <script charset="UTF-8" type="application/javascript" src="path/ke/lokasi/javascript.js"></script>
  </head>
</html>

Daftar isi

Statement

Program javascript dieksekusi baris perbaris. Artinya, dalam memproses file javascript, browser akan membaca baris-perbaris source code dan langsung dieksekusi seketika itu juga. Contoh:

const a = 1;
const b = 2;
const c = a + b;
alert(c);

Program tersebut akan menampilkan dialog dengan isi 3.

Setiap baris program dalam javascript ditutup dengan tanda ;. Dalam peristilahan pemrograman, baris code yang dieksekusi oleh program disebut dengan istilah statement.

Daftar isi

Output

Output atau hasil program javascript dapat ditampilkan dengan berbagai cara. Untuk pemula, agar lebih muda diobservasi, umumnya output javascript dilakukan dengan menggunakan alert sebagaimana di atas. Akan tetapi, penggunaan alert sebagaimana di atas dapat mengganggu penggunaan website.

Cara lain yang lebih tidak mengganggu adalah menggunakan console.log. Contohnya:

const a = 1;
const b = 2;
const c = a + b;
console.log(c);

Untuk melihat hasil program di atas, harus membuka developer tools terlebih dahulu.

Variables

Variable digunakan untuk memberi nama pada suatu data agar mudah diingat oleh penulis program. Pembuatan variable dapat dilakukan dengan tiga cara, yaitu:

  • const c = 3;
  • let b = 2;
  • var a = 1;

Perbedaannya, const digunakan untuk menamai variable yang tidak akan diganti lagi. Contohnya:

const a = 1;
a = 10;

Program di atas akan error, karena program tersebut mencoba merubah variable a yang asalnya berisi 1 menjadi 10. Berbeda dengan let atau var, keduanya mengizinkan perubahan data/isi variable. Contoh:

let a = 1;
var b = 2;
a = 10;
b = 20;

Program tersebut dapat berjalan tanpa masalah.

Karena const tidak dapat dirubah, maka disarankan untuk menggunakan const selama tidak ada rencana perubahan isi variable.

Data Type

Dalam Javascript, ada setidaknya 3 data type primitif, yaitu: boolean, number dan string. Contohnya: 


const var1 = true;    // boolean
const var2 = 10;      // number
const var3 = '10';    // string

Daftar isi

boolean

boolean adalah data type digunakan untuk menunjukkan benar (true) atau salah (false). Contohnya: 


const var1 = 10;
const var2 = 30;
const var3 = var1 === var2;
console.log(var3);

Daftar isi

number

number adalah data type yang digunakan untuk menunjukkan nilai angka sehingga dapat dioperasikan menggunakan operator aritmetik. 


const var1 = 10;
const var2 = 30;
const var3 = var1 + var2;
console.log(var3); // 40

Daftar isi

string

string adalah data type yang digunakan untuk menunjukkan teks atau tulisan. string ditulis dengan tanda kutip satu ' atau tanda kutip dua ". 


const var1 = 'variable 1';
const var2 = "variable 2";
const var3 = var1 + var2;
console.log(var3); // variable 1variable 2

Kemudian, ada 2 data type komposit yang perlu diketahui, yaitu: array dan object.

Daftar isi

array

array adalah data type yang digunakan untuk menampung beberapa data sekaligus. Penulisan array, harus dibuka dengan [ dan ditutup dengan ]. Isi satu persatu array dikenal dengan sebutan element, yang urutannya dimulai dari 0. 


const cities = [
    'Cilacap',
    'New York',
    'London',
    'Lampung',
];
console.log(cities);

Element dalam array diakses dengan cara menuliskan tanda [, kemudian nomor urut element dan ditutup dengan tanda ]. Contoh: 


const cities = [
    'Cilacap',
    'New York',
    'London',
    'Lampung',
];
console.log(cities[0]); // Cilacap
console.log(cities[1]); // New York
console.log(cities[2]); // London
console.log(cities[3]); // Lampung

Daftar isi

object

object adalah data type yang digunakan untuk menampung pasangan data dan labelnya (property). Penulisan object, diawali dengan menulis tanda {, diikuti dengan property yang ditulis dengan menuliskan labelnya, dilanjutkan dengan tanda : kemudian isi data. property dipisah dengan menggunakan tanda ,. Contoh: 


const user = {
  name: 'Ahmed',
  age: 24,
  phone: '081234567890',
  email: 'ahmed@blkk.com'
};

Contoh di atas adalah contoh object dengan empat property, yaitu: name, age, phone dan email.

Property dalam object diakses dengan cara menuliskan tanda [, kemudian string nama property dan ditutup dengan tanda ]. Contoh: 


const user = {
  name: 'Ahmed',
  age: 24,
  phone: '081234567890',
  email: 'ahmed@blkk.com'
};
console.log(user['name']);
console.log(user['age']);
console.log(user['phone']);
console.log(user['email']);

Property milik object juga dapat diakses dengan cara lebih simpel, yaitu dengan menuliskan tanda . kemudian nama propertynya. Cara ini hanya digunakan ketika labelnya hanya tersusun dari huruf-huruf. Contoh: 


const user = {
  name: 'Ahmed',
  age: 24,
  phone: '081234567890',
  email: 'ahmed@blkk.com'
};
console.log(user.name);
console.log(user.age);
console.log(user.phone);
console.log(user.email);

Function

function adalah sekumpulan statement yang dikumpulkan menjadi satu sehingga dapat digunakan berulang-ulang.

Perhatikan contoh berikut:


const users = [
  { name: 'Ahmed', age: '24' },
  { name: 'Jack', age: '32' },
  { name: 'John', age: '35' },
];
let text = 'Nama: ' + users[0].name + ', Usia: ' + users[0].age;
console.log(text);
text = 'Nama: ' + users[1].name + ', Usia: ' + users[1].age;
console.log(text);
text = 'Nama: ' + users[2].name + ', Usia: ' + users[2].age;
console.log(text);

Code di atas melakukan penggabungan teks yang tersusun dari

  1. 'Nama: '
  2. property name dari object ke n yang terdapat di array users
  3. ', Usia: '
  4. property age dari object ke n yang terdapat di array users

Dalam code di atas, penggabungan string dilakukan sampai tiga kali. Padahal, pada prinsipnya yang dilakukan ketiga statement tersebut sama saja. Pengulangan code semacam ini tidak disarankan karena jika ada perubahan code, kita harus merubah satu per satu dari tiga statement tersebut.

Untuk itu, pengulangan code dapat disimplifikasi dengan membuat function. Contoh: 


function debugUser(user) {
  let text = 'Nama: ' + users[0].name + ', Usia: ' + users[0].age;
  console.log(text);
}

const users = [
  { name: 'Ahmed', age: '24' },
  { name: 'Jack', age: '32' },
  { name: 'John', age: '35' },
];
debugUser(users[0]);
debugUser(users[1]);
debugUser(users[2]);

Pada contoh di atas, penggabungan string hanya dilakukan di satu tempat. Jadi, seandainya ada perubahan code, yang perlu dirubah hanyalah badan functionnya.

Penulisan function harus diawali dengan kata kunci function, diikuti dengan tanda kurung yang memuat daftar parameter yang dipisah dengan koma, kemudian diikuti dengan bada function yang dimuat di dalam tanda {} (kurung kurawal). 


function namaFunction(param1, param2, ...paramLainnya) {
  // daftar statement
}

Lalu, untuk menggunakan function yang sudah didefinisikan. Tuliskan nama function diikuti dengan tanda kurung (()) yang memuat daftar parameter. Contoh: 


const a = 0;
const b = 1;
const c = 2;
const d = 3;
const e = 4;
const f = 5;
namaFunction(a, b);
namaFunction(c, d);
namaFunction(e, f);

Dalam pemrograman, menggunakan function disebut dengan function call. Ketika function call berlangsung, setiap parameter yang ada di definisi function akan diganti dengan data yang diberika saat function call. Misalkan ada definisi function sebagaimana berikut: 


function fungsiContoh(a, b) {
  console.log(a);
  console.log(b);
}

Jadi, ketika ada statement seperti di bawah ini:

fungsiContoh(1, 2);

Maka, parameter a dalam fungsi di atas, akan berubah menjadi 1 dan parameter b akan berubah menjadi 2.

Return Statement

Sering kali, function digunakan untuk melakukan kalkulasi kompleks yang mana hasilnya diperlukan untuk proses kalkulasi lebih lanjut. Untuk mengeluarkan hasil kalkulasi dari sebuah function, digunakan kata kunci return diikuti dengan data yang memuat hasil kalkulasi di dalam function yang bersangkutan. Contohnya: 


function penjumlahan(a, b) {
  return a + b;
}

const c = penjumlahan(1, 2);

Function tersebut melakukan kalkulasi penjumlahan yang kemudian hasilnya dikembalikan ke luar konteks function tersebut yang mana dalam contoh di atas dimasukkan ke dalam variable c.

Dengan demikian, kita dapat membuat function yang mana menggunakan function yang lain. Contoh: 


function double(angka) {
  return angka * 2;
}

function kuadrat(angka) {
  return angka * angka
}

function kalkulasiInternal(angka) {
  const a = double(angka);
  return kuadrat(a) + 1;
}

function kalkulasiRumit(angka) {
  return kalkulasiInternal(angka) + kalkulasiInternal(angka);
}

const hasil = kalkulasiRumit(1);

console.log(hasil); // Tebak, apa output nya?

Method

Method adalah sebuah function yang menjadi property dari sebuah object. Contoh: 


const user = {
  name: 'Jack',
  sayHello: function() {
    const text = 'Hello, my name is ' + this.name;
    console.log(text);
  }
};

user.sayHello();

Method dan function pada prinsipnya sama. Perbedaannya adalah:

  • Penulisan method dapat dilakukan tanpa menuliskan nama functionnya sebagaimana contoh di atas.
  • Penulisan method dapat dituliskan langsung sebagai property dari sebuah object sebagaimana contoh di atas ataupun sebagai variable terpisah.
  • Di dalam method terdapat variable khusus this yang dapat digunakan untuk mengakses property object yang bersangkutan.
  • Penggunaan Method method dilakukan dengan mengakses method tersebut sebagai property kemudian dilakukan function call.

Contoh penulisan method sebagai variable terpisah. 


function namaFunction() {
  const text = 'Hello, my name is ' + this.name;
  console.log(text);
}

const user = {
  name: 'Jack',
  sayHello: namaFunction
}

user.sayHello();

Conditional

Terkadang sebuah program yang perlu dieksekusi ketika memenuhi kriteria tertentu. Eksekusi program yang bergantung pada suatu kriteria disebut conditional statement. Conditional statement dituliskan dengan menggunakan kata kunci if, diikuti dengan tanda kurung (()) yang memuat syarat/kondisi, diikuti dengan tanda kurung kurawal ({}) yang memuat block kode. Contoh: 


if (a < 10) {
  console.log(a, ' lebih kecil dari 10');
}

Pada contoh di atas, kode yang berada di dalam kurung kurawal ({}) hanya dieksekusi jika isi a adalah angka yang lebih besar dari 10.

Conditional statement dapat juga memuat alternatif kode yang akan dijalankan ketika syarat yang terdapat di if tidak terpenuhi. Caranya adalah dengan menambahkan kata kunci else setelah tutup kurawal (}) milik if. Contoh: 


if (a < 10) {
  console.log(a, ' lebih kecil dari 10');
} else {
  console.log(a, ' lebih besar dari 10');
}

Alternatif dari conditional statement bisa lebih dari satu cabang dengan menggunakan else if yang dituliskan sebagaimana if namun dituliskan sesudah block if dan sebelum block else (jika block else diperlukan). Contoh: 


if (a < 10) {
  console.log(a, ' lebih kecil dari 10');
} else if (a < 20) {
  console.log(a, ' lebih kecil dari 20');
} else if (a < 30) {
  console.log(a, ' lebih kecil dari 30');
} else if (a < 40) {
  console.log(a, ' lebih kecil dari 40');
}

Contoh tanpa block else:


if (a < 10) {
  console.log(a, ' lebih kecil dari 10');
} else if (a < 20) {
  console.log(a, ' lebih kecil dari 20');
} else if (a < 30) {
  console.log(a, ' lebih kecil dari 30');
} else if (a < 40) {
  console.log(a, ' lebih kecil dari 40');
} else {
  console.log(a, ' lebih besar dari 40');
}
